# whatToEat
使用Struts和hibernate建立"今天吃什么"网站,让决定吃什么不再纠结~~

## 项目背景
因为自己每天都要纠结于吃什么的问题,面对着众多选择无所适从=.=又出于锻炼自己对于SSH框架的使用的目的,因此选择建立一个名为"今天吃什么"的网站,让你下一顿吃什么不再纠结~~~~

## 网站基本功能:
* 用户注册登录
* 菜品推荐
* 用户可以选择收藏菜品
* 当然啦用户也可以把一些菜品放入黑名单

## 理想的高级功能=.=
* 对用户喜好的推测
* 根据用户的喜好进行智能推荐
* 菜谱推送
